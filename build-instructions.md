# Build instructions

- clone the repository

- Run `go build -o learning-go github.com/bijeshos/learning-go`
    - to build the project

- Run `go install -o learning-go github.com/bijeshos/learning-go`
    - to install the binaries

- Run `./learning-go`
    - to run the binary

- Run `go run main.go`
    - to run main program
